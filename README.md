## Keyboard Controls

- **Space**: Start/Pause game
- **R**: Reset game
- **A**: Toggle auto-play

## Setup

1. Extract all files into a folder
2. Open `index.html` in a web browser
3. No server required - works directly in browser

## Game Rules

- Standard 5x5 bingo card
- Center space is FREE
- Numbers 1-75
- Win with any row, column, or diagonal

## File Structure

- `index.html` - Main game file
- `styles/` - CSS styling and animations
- `scripts/` - JavaScript game logic
- `assets/` - Images and audio files
